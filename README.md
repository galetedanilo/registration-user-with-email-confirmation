# Como Executar o Projeto

## Back end:

Pré-requisito: Java 11

```bash
# clonar repositório
git clone https://github.com/galetedanilo/registration-user-with-email-confirmation.git

# entrar na pasta do back end
cd backend

# executar projeto
./mvnw spring-boot:run
```
